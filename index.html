Everything runs in your browser — images never leave this device — and the progress log above streams while pages are processed. In the results, a green/blue/red number is a confident reading, * marks a note read via a detected accidental, and an orange number is a best guess — verify it by eye. A detected note with no number printed has no slide position in the chosen method (it appears as ? in the log); the space is left blank so you can pencil it in. When the key is auto-detected, the log shows a detected key line per page — if it disagrees with the printed signature, pick the key manually and run again.
