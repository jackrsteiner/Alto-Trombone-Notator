Everything runs in your browser — images never leave this device. In the results, a green/blue/red number is a confident reading, * marks a note read via a detected accidental, and an orange number or ? means “verify this one by eye”. When the key is auto-detected, the log shows a detected key line per page — if it disagrees with the printed signature, pick the key manually and run again.
